# AGENTS.md


## Code style
- Functions: camelCase (`getUserData`, `calculateTotal`)
- Variables: camelCase (`userData`, `totalCurrency`)
- Classes: PascalCase (`UserService`, `DataController`)
- Constants: UPPER_SNAKE_CASE (`API_KEY`, `MAX_RETRIES`)
- Always use descriptive names when defining functions, classes, variables and constants
- Use the MVVM framework.


## Editing files
- Make the smallest safe change that solves the issue.
- Preserve existing style and conventions.
- Prefer patch-style edits (small, reviewable diffs) over full-file rewrites.
